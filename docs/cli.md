# CLI Reference

| Command      | Description                                 |
|--------------|---------------------------------------------|
| `s <ID>`     | Show details of a service                   |
| `p <ID>`     | Show all processes of a service             |
| `k <ID>`     | Kill a service's main PID                   |
| `K`          | Kill all services                           |
| `t <ID>`     | Stop/deactivate a service                   |
| `T`          | Stop all services                           |
| `r`          | Refresh the list                            |
| `q`          | Quit the tool                               |

See [Usage](./usage.md) for more details.
